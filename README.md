<p>

（少なくとも標準ライブラリであるMathクラスの中には存在しない。）<br>
 そこで、自身の開発時に冗長なコードを繰り返し記述することを避けるべく、<br>
  汎用性が高く、それをコピー&ペースト、あるいは数的処理用クラスとしてプログラムに組み込むことを前提としたコードを制作した。
  </p>
